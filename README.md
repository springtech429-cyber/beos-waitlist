# BEOS Waitlist

The Operating System for the Built Environment — Waitlist Page

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Cloudflare Pages

This project is configured for Cloudflare Pages. The build output goes to `dist/`.

- **Build command:** `npm run build`
- **Build output directory:** `dist`
- **Root directory:** `/`
